# UI 设计规范 (UI Design System)

本文档概述了 Friend Poker Game 项目的核心 UI 参数、颜色和布局原则。任何新的前端设计或 UI 修改都应符合这些准则。

## 1. 全局主题与背景
- **应用背景 (Canvas)**：`#0b0f1a`（深午夜蓝/黑色）。
- **整体文字颜色**：基本可读文本使用 `text-slate-300`，强调内容过渡为 `text-white`。

## 2. 游戏牌桌
核心扑克牌桌采用多层嵌套椭圆设计，以营造深度和真实的“毛毡”桌面感。
- **外层椭圆（桌面基础）**：`#2e3a4d`
  - 圆角半径：移动端 `rounded-[160px]`，桌面端 `rounded-[220px]`。
  - 使用阴影增加深度 (`shadow-[0_20px_50px_rgba...`)。
- **内层椭圆（游戏区域）**：`#3e4f67`
  - 圆角半径：移动端 `rounded-[135px]`，桌面端 `rounded-[180px]`。
- **内边界/线条**：`#2b3749`

### 底池与公共牌区域
- **公共牌容器**：`bg-black/20`，带有微妙的边框 `border-black/30`。
- **底池 (Pot) 显示**：`bg-[#242f40]/90` 加上金色文字 (`text-amber-400`)。

## 3. 扑克牌显示
- **扑克牌尺寸**：
  - 默认/移动端：`w-[48px] h-[68px]`
  - 桌面端：`w-[60px] h-[85px]`
- **花色颜色**：红心/方块使用 `text-red-600`，梅花/黑桃使用 `text-slate-900`。
- **底牌（玩家手牌）扇形样式**：通过边距重叠 (`-space-x-[20px]`) 和旋转 (`-rotate-[5deg]` 以及 `rotate-[5deg]`) 展示。

## 4. 玩家名牌 (Player Nameplates)
玩家信息框显示玩家姓名、筹码/状态，并在牌桌周围作为视觉锚点。
- **尺寸**：固定高度 `h-[48px]`，最小宽度 `min-w-[100px]`。
- **背景**：`bg-black/90` 加上圆角边缘。
- **活跃状态（当前回合）**：当轮到该玩家时，边框发光 `border-indigo-500` 并伴随阴影 `shadow-[0_0_20px_rgba(99,102,241,0.6)]`。包含 `scale-110` 放大效果以吸引视线。
- **英雄状态（自己）**：当前本地玩家获得白色外发光 `border-white/80`。
- **庄家按钮 (Dealer Button)**：24x24 像素的圆形，使用金色渐变 (`from-[#fde047] to-[#f59e0b]`)。

## 5. 悬浮操作按钮（玩家操作区）
玩家操作按钮固定在屏幕的右下角，而不是占用整个底部宽度，确保牌桌保持可见且不被遮挡。
- **高度**：主要操作按钮（FOLD / CALL / RAISE）高度统一：移动端 `h-[52px]`，桌面端 `h-[60px]`。
- **颜色**：
  - **FOLD (弃牌)**：背景 `bg-rose-900/30`，边框 `border-rose-500/40`，文字 `text-rose-300`。
  - **CALL/CHECK (跟注/过牌)**：背景 `bg-slate-800/80`，边框 `border-slate-600/60`，文字 `text-slate-200`。
  - **RAISE (加注)**：背景 `bg-indigo-600/90`，边框 `border-indigo-500`，文字 `text-white`。
- **快捷加注按钮**：位于操作栏上方的三个快捷按钮，用于快速加注底池比例（50%、75%、100%），采用小型药丸按钮样式（`px-3 py-1.5 bg-slate-800/80`）。

## 6. 房主控制面板与初始登录区
- **主要按钮 (创建/加入/开始)**：使用 `bg-indigo-600` 以增强视觉焦点。
- **房主控制容器面板**：包裹在 `bg-slate-900/80` 中并深度模糊 (`backdrop-blur-md`)。为了在作为房主时节省屏幕空间，面板高度受到压缩，对于玩家列表使用带有滚动条的限制高度面板 (`max-h-[250px]`)。
